## Run the App

First, change the directory by running the command:
```sh
cd calculator-logic
```

In the calculator-logic directory, you can run: 
```sh
open index.html
```

If you want to run tests:
  ```sh
  //if your package manager is:
    //npm
    npm test
    npm t

    //yarn
    yarn test
  ```


## Information about the project

Calculator app is a simple calculator for general purposes. You can use it to multiply, divide, add and subtract numbers. 

### Features:
* multiplication
* division
* addition
* substraction
* all clear (AC)
